# TikTok App Description

App name:
Social Video Poster

Short description:
A desktop scheduling tool that lets creators and account owners upload their own local videos to their own TikTok accounts through TikTok Login Kit and the Content Posting API.

Intended use:
- The user connects their own TikTok account.
- The user provides video files they own or control.
- The app uploads scheduled posts to that authorized account.

Why TikTok access is needed:
- `video.publish` is required to upload user-approved videos to the user's own TikTok account.
- Basic user authorization is required so the app can publish only after explicit user consent.

What the app does not do:
- It does not scrape TikTok.
- It does not access accounts without explicit authorization.
- It does not publish third-party content without user action.
