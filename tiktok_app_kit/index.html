TikTok Content Posting Integration

Social Video Poster

Social Video Poster is a desktop automation tool that helps authorized users publish original short-form videos from a local media library to their own social accounts.

The app uses TikTok Login Kit and Content Posting API to let creators connect their own TikTok account and upload videos they own or control. Users explicitly authorize access before any upload happens.

Core Function

Queue local videos and publish them to an authorized TikTok account on a schedule.

User Control

Only the account owner can authorize publishing access for their TikTok account.

Content Scope

The app is intended for original creator-owned content and simple repost scheduling.
